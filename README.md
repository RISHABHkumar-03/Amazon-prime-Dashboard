# Amazon-prime-Dashboard

1. **The title of project is** :
Amazon Prime Dashboard(Movies & TV.shows)

2. **Purpose / Description**:
This dashboard provides an overview of Amazon Prime Video content by analyzing movies and TV shows based on ratings, genres, countries, and release trends. It helps understand content distribution, audience focus, and growth patterns over time.

3. **Tech Stack:**:
> Power BI – for data visualization and interactive dashboard creation
> Power Query – for data cleaning and transformation
> DAX (Data Analysis Expressions) – for calculated measures and KPIs
> Dataset (CSV/Excel) – as the data source for **movies **and **TV shows**
> Microsoft Bing Maps – for country-wise geographical visualization

4. **Data Source:**
The dashboard uses the Amazon Prime Video dataset, stored in a structured tabular format (CSV/Excel). Each row represents a single movie or TV show, identified by a **unique show_id**. The dataset includes attributes such as **title**,** type** (Movie or TV Show), **release year**, **rating**, **duration**, **director**, **country**, **genre** (listed_in), and **description**, making it suitable for analyzing content distribution, trends, and audience targeting in Power BI.

5. Features / Highlights
**Comprehensive Content Overview**
Displays total titles, genres, ratings, directors, and release timeline, giving a quick snapshot of Amazon Prime Video’s content library.
**Rating-Based Content Analysis**
Shows how content is distributed across different age ratings, helping identify the platform’s target audience.
**Genre Popularity Insights**
Highlights the most common genres such as Drama and Comedy, making it easy to understand viewer content preferences.
**Geographical Distribution**
Interactive world map visualizes country-wise content availability, identifying major content-producing regions.
**Movies vs TV Shows Comparison**
Donut chart clearly compares the proportion of movies and TV shows for quick content-type analysis.
**Release Year Trend Analysis**
Line chart shows growth in content over time, revealing rapid expansion in recent years.
**Interactive & User-Friendly Design**
Filters and visuals allow users to explore data dynamically for deeper insights.
